# C++ containers

## О программе
  Библиотека копирующая (неполную) реализацию методов стандартных контейнеров c++:
 - array
 - list
 - map
 - set
 - multiset
 - queue
 - stack
 - vector

  map, set, multiset основаны на красно-черном дереве.
